# Business Use Case: Omnichannel Customer Events

A retailer wants one scalable data platform for behavioral analytics.

- Hadoop demonstrates distributed batch counting.
- Spark creates curated Parquet datasets.
- Hive gives analysts SQL access.
- Pig demonstrates declarative data-flow ETL.
- HBase supports low-latency event or customer lookups.
- Spark K-Means creates behavioral clusters.
- S3 provides durable, low-cost lake storage.
- EMR supplies ephemeral managed compute.
